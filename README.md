# CHIP-8
CHIP-8 interpreter implementation in C.

run `make` command to compile

in order to run:
`./chip <Scale> <Delay> <ROM>`

```
Keyboard     CHIP-8
+-+-+-+-+    +-+-+-+-+
|1|2|3|4|    |1|2|3|C|
+-+-+-+-+    +-+-+-+-+
|Q|W|E|R|    |4|5|6|D|
+-+-+-+-+    +-+-+-+-+
|A|S|D|F|    |7|8|9|E|
+-+-+-+-+    +-+-+-+-+
|Z|X|C|V|    |A|0|B|F|
+-+-+-+-+    +-+-+-+-+
```
